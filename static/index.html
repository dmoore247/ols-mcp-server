OLS MCP Server on Databricks Apps

A Model Context Protocol server for the Ontology Lookup Service (OLS) API, giving AI assistants access to biomedical ontologies.

View Tools Connect

Available Tools

search_terms

Search for terms across ontologies using the OLS search API. Supports exact matching, obsolete filtering, and ontology restriction.

get_ontology_info

Get detailed information about a specific ontology by its identifier (e.g., 'efo', 'go', 'mondo').

search_ontologies

Search for available ontologies with optional text filtering and pagination.

get_term_info

Get detailed information about a specific term by IRI, short form, or OBO ID.

get_term_children

Get direct children of a specific term within an ontology hierarchy.

get_term_ancestors

Get ancestor terms (parents) of a specific term in the ontology graph.

find_similar_terms

Find terms similar to a given term using LLM embeddings for semantic similarity.

Connect to the Server

Streamable HTTP
Example Connection (Python)

from databricks.sdk import WorkspaceClient
from databricks_mcp import DatabricksMCPClient

# Initialize the Databricks Workspace client
ws_client = WorkspaceClient(
    host="databricks-host",
    token="exactly-scoped-OAuth-token",
)

# Initialize the MCP client with your app's MCP endpoint
mcp_client = DatabricksMCPClient(
    server_url='databricks-app-url',
    workspace_client=ws_client,
)

# List available MCP tools
print(mcp_client.list_tools())
